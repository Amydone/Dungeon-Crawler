# Dungeon-Crawler
Simple console game based on Windows (x64) platform



## Requirements 

> g++ compiler or another, i've used Microsoft C++ Compiler [MSVC]



## Compilation

> For g++ --v 9 and later
```
g++ -std=c++2a <files>.cpp -o <name>.exe
```
> For g++ --v 8 and late
```
g++ -std=c++20 <files>.cpp -o <name>.exe
```
